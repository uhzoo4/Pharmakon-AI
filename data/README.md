# Pharmakon – Training Data Directory

Place your raw training corpus files (.txt) here to train the corresponding literary personalities.

### File Requirements:
- **Format:** Plain text files ending with `.txt`.
- **Naming Convention:** Match the personality key exactly:
  - `kafkaesque.txt` for the Kafkaesque Bureaucrat.
  - `camus_stranger.txt` for the Camus Absurdist.
  - `dark_romance.txt` for the Dark Romance.
- **Encoding:** Strictly UTF-8 encoded text files.
- **Size:** Ensure you have enough corpus text (recommended $>10,000$ characters of prose) for standard character-level training.
