## Closest real match in the style library
Checked all 20 web philosophies in `references/design-styles.md` rather
than guessing. The closest fit isn't generically "cinematic dark" — it's
**复古未来太空图录 / Cosmic Retro-Futurism** (Bold school, listed ~75%
fidelity in pure HTML/CSS+SVG):

- Referenced mood: *2001: A Space Odyssey*, old astronomical atlas plates
- Palette: near-black `#0A0A0A`, cream/paper `#F0EAD8`, cobalt-peacock
  blue `#2B4F91` — desaturated, like aged star-chart prints, not
  screen-saturated sci-fi
- Typographic DNA: high-contrast classical serif (the library suggests
  Cormorant Garamond / EB Garamond) against generous whitespace, "old
  astronomy book" register rather than a tech-product sans
  Structure: SVG orbital paths and parabola lines, planetary dots,
  cream ground under black type, archival print layout logic
- The skill's own honest fidelity note: pure CSS+SVG gets you ~80% of the
  static mood; the gap is the full-screen "object falling from space to
  earth" video transition, which it says needs to degrade to scroll
  parallax + rotating SVG orbits rather than true video

For a myth about a flight path — ascent, a warning, a climb toward the
sun, a fall — an astronomical-plate visual language (orbit lines, a
parabolic arc, a sun as a literal graphic element) is a structurally
honest match, not just a mood-board coincidence. Worth actually reading
the full entry in `references/design-styles.md` before committing; this
is a summary, not the complete spec.

## Secondary option, worth a look
**电影感声波可视化 / Cinematic Sound-Viz Dark** (also Bold school): pure
black ground, Saul Bass-style minimal title-sequence typography, a
waveform/frequency motif running through the layout. Less myth-specific
than the retro-futurism option, but the "film title sequence" register
overlaps with the cinematic goal directly if the orbital motif feels too
literal for any given scene.

## Mood keywords (for the skill's own intake questions)
mythic, archival, astronomical, precise, atmospheric, high-contrast,
paper-and-ink-on-black, ascent-and-fall

## Usage scenario
Web (8-scene scroll narrative), not PPT/print — route to the web style
library only, not the PPT one, despite the "poster" quality of the
reference mood.

This is one match out of 20 real entries, picked by reading the actual
